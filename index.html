Overview

What is this project?

This project simulates a real-world pencil manufacturing production line as part of the Applied Mechatronics Advanced Programming module at SRH Fernhochschule. The simulation models four sequential assembly stages, applies probabilistic defect detection at each stage, streams all metrics to InfluxDB, and visualises them in both a custom HMI and a Grafana dashboard.

Production Line

Four Assembly Stages

Each stage is processed sequentially. A unit is immediately flagged defective if any stage fails — no further assembly is performed on a known-bad unit.

1

Graphite Core

Graphite–clay rod inserted and pressed into a pre-routed cedar wood slot.

⚠ 5% defect rate

Failure: core fracture during insertion

2

Pencil Body

Two cedar halves glued, clamped, then shaped and sanded to final profile.

⚠ 4% defect rate

Failure: body split — wood grain defect

3

Eraser

Synthetic rubber eraser plug inserted into the ferrule seat at the pencil top.

⚠ 6% defect rate

Failure: eraser misaligned or missing

4

Eraser Holder

Metal ferrule crimped around the eraser plug to lock it permanently in place.

⚠ 3% defect rate

Failure: crimp failure — loose ferrule

Architecture

Four-Container Stack

The entire application runs as four Docker containers orchestrated by Docker Compose. Each container has a single, clearly-defined responsibility.

🐍

Flask Backend

REST API with production logic. Runs the defect model, writes metrics to InfluxDB.

:5000 — pencil_backend
🗄

InfluxDB 2.7

Time-series database storing 9 fields per production tick including temperatures and yield.

:8086 — pencil_influxdb
📊

Grafana 10.2

Auto-provisioned dashboard via bind-mounted provisioning folder. No manual setup needed.

:3000 — pencil_grafana
🌐

Nginx + HMI

Serves the static HTML/JS operator interface with Chart.js live charts.

:80 — pencil_hmi

Getting Started

Run the Simulation

Prerequisites: Docker Desktop installed and running. No other dependencies needed.

# 1. Clone the repository
git clone https://github.com/mshawky772007-eng/Advanced-Programming-Project
# 2. Enter the project folder
cd pencil-production/pencil
# 3. Start all containers
docker compose up --build
# 4. Open the HMI
→ http://localhost (HMI operator interface)
→ http://localhost:3000 (Grafana dashboard — admin/admin)
→ http://localhost:5000 (Flask API)
